# Blog


## Lokalní spuštění

```
npm install
bundle install --path vendor/bundle
bundle exec jekyll serve --livereload --future --drafts
```

## Install

On Fedora 27:

```
dnf in gcc rubygem-bundler ruby-devel rubygem-jekyll rubygem-nokogiri
bundle install --path vendor/bundle
bundle exec jekyll serve --livereload --future --drafts
```
